## MCP 서버 설치

```js
npm install -g @modelcontextprotocol/server-filesystem
```


## 파일시스템 서버 설정

```js
{  
    "mcpServers": {  
        "filesystem": {  
        "command": "npx",  
        "args": [  
            "@modelcontextprotocol/server-filesystem",  
            "D:\\MCP_Example\\Claude_Memos"  
        ]  
        }  
    }  
}  
```
